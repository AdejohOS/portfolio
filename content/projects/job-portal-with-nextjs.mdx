A fully functional Job search application, built with Next 14, Typescript,
Auth.js, Prisma, Shadcn Ui, Uploadthing, Ky, Sql database, and Tanstack Query.

[GitHub Repo](https://github.com/AdejohOS/futurejobs)

[Live Site](https://futurejobs.vercel.app/)

## Features

- **Authentication**: User creation, authentication and management.
- **Cloud Storage**: Storing assets such as our images & files.
- **Mutations**: Mutation functions such as job application with optimistic ui.
- **Infinite Scrolling**: More job posts reveal as we scroll down, typically a
  social media feeds concept replacing pangination in this application.
- **Server Actions**: Form submission was handled with server actions introduced
  in Next 13.
- **Responsive Design**: The website is responsive and works on all devices.

## Technologies

- **[NextJs](https://nextjs.org/)**: A React framework for building static and
  dynamic websites.
- **[Auth.js](https://authjs.dev/)**: User creation, authentication and
  management with .
- **[Uploadthing](https://uploadthing.com/)**: Storing our assets such as images
  & files.

- **[Tanstack React Query](https://tanstack.com/query/latest)**: Was used for
  mutation functions such as infinite scrolling and fetching data.
- **[Tailwind CSS](https://tailwindcss.com/)**: A utility-first CSS framework
  for building custom designs.
- **[Shadcn Ui](https://tailwindcss.com/)**: A customizable component library
  built off Tailwind for React and Next.js aplications.
- **[Ky](https://www.npmjs.com/package/ky)**: A lightweight HTTP client for
  making network request.
- **[Prisma ORM](https://next-auth.js.org/v3/adapters/prisma)**: A javascript
  and typescript ORM (Object Relational Mapping) designed to make database
  interactions easier and type-safe.
- **[Sql database](https://aiven.io/)**: Structured Query Language (SQL) is a
  relational database where data are arranged in tables which consist of rows
  and columns.

## Getting Started

To get started with this project, you can clone the
[repository](https://github.com/AdejohOS/futurejobs) and install the
dependencies:

```bash
git clone repo
cd futurejobs
npm install
```

Once the dependencies are installed, you can create a .env to add you
enviromental variables from Database, Github, Google and Uploadthing. You can
now run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.

## Deployment

To deploy this project, you can use Vercel, create a virtual server or whatever
hosting option you choose.

## Conclusion

The futurejobs project is a great example of how you can use NextJs 14 and a
wide variety of libraries such as Tanstack Query, Prisma, AuthJs to build a
modern website with dynamic content.
